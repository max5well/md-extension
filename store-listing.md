## Detailed Description

**Stop copy-pasting messy web content into your AI.**

Every time you paste a webpage into ChatGPT, Claude, or Gemini — you're sending noise. Ads. Nav menus. Cookie banners. Garbage the AI has to wade through before getting to what you actually care about.

Copy as Markdown fixes that. One click. Clean Markdown. Done.

────────────────────────────

**HOW IT WORKS**

1. Go to any webpage
2. Click the Copy as Markdown icon
3. Paste into ChatGPT, Claude, Notion, Obsidian — anywhere

That's it. No settings. No accounts. No nonsense.

────────────────────────────

**WHAT YOU GET**

✓ One-click copy — click the icon, your clipboard is ready
✓ AI-ready output — clean Markdown that LLMs actually understand
✓ Smart extraction — automatically finds the main content, ignores nav, ads, sidebars, and footers
✓ Structured output — page title becomes a # heading, source URL is included
✓ Works everywhere — articles, docs, blogs, Wikipedia, GitHub, Reddit, and more
✓ 100% private — no servers, no accounts, no data ever leaves your browser

────────────────────────────

**WHY MARKDOWN FOR AI?**

LLMs read Markdown natively. When you paste raw HTML or plain text, you lose structure — headings collapse, lists break, code blocks disappear. That means worse answers.

Markdown keeps everything intact: headings, bullet points, code, tables, and links. Your AI gets better context. You get better answers. It's that simple.

────────────────────────────

**PERFECT FOR**

→ AI power users who paste web content into ChatGPT, Claude, or Gemini daily
→ Notion and Obsidian users building a second brain
→ Researchers and writers collecting information from the web
→ Developers copying documentation into their notes
→ Anyone who's ever screamed at a messy clipboard paste

────────────────────────────

**PRIVACY**

Copy as Markdown never collects, stores, or transmits your data. All processing is local. The extension only activates when you click the icon. Nothing is sent anywhere — ever.

────────────────────────────

**PERMISSIONS EXPLAINED**

• activeTab — to read the current page when you click the icon
• scripting — to extract and convert the page content
• clipboardWrite — to write the Markdown to your clipboard

No other permissions. No background tracking. No analytics.

────────────────────────────
