## Name
Copy as Markdown

## Short Description (132 chars max)
Instantly copy any webpage as clean Markdown — perfect for pasting into ChatGPT, Claude, Notion, or your notes.

## Detailed Description

**Turn any webpage into Markdown with one click.**

Copy as Markdown extracts the content of any webpage and copies it as clean, formatted Markdown to your clipboard. No clutter, no ads, no sidebars — just the content.

**Perfect for:**
- Pasting articles into ChatGPT, Claude, Gemini, or any LLM chat
- Saving research into Obsidian, Notion, or Roam
- Copying documentation into your notes
- Feeding web content to AI tools without losing formatting

**What you get:**
✓ One-click copy — click the icon, done
✓ Clean output — strips nav, ads, headers, and footers automatically
✓ Smart extraction — prefers `<main>` and `<article>` content
✓ Structured output — page title as `# Heading`, source URL included
✓ Works on any page — articles, docs, blogs, Wikipedia, GitHub, and more
✓ Fully local — no servers, no accounts, no data collection

**Why Markdown?**
LLMs understand Markdown natively. Pasting a raw webpage into an AI chat loses structure and wastes tokens. Pasting Markdown keeps headings, lists, code blocks, and links intact — so your AI gets better context and you get better answers.

## Privacy Policy Summary
Copy as Markdown does not collect, transmit, or store any user data. All processing happens locally in your browser. The extension only reads page content when you explicitly click the icon, and writes exclusively to your local clipboard.
