Privacy Policy

Last updated: April 2, 2026

The short version

Copy as Markdown does not collect, store, or transmit any data. Ever. All processing happens locally in your browser.

What we collect

Nothing. Copy as Markdown has no backend, no database, no analytics, and no tracking code.

How the extension works

When you click the extension icon, it reads the current page's HTML content, converts it to Markdown using a local library, and writes the result to your clipboard. This entire process happens inside your browser tab. No data is sent to any external server.

Permissions explained

activeTab — Used to access the current page when you explicitly click the icon. The extension cannot access any page you haven't activated it on.

scripting — Used to inject the conversion script into the active tab to extract and convert content.

clipboardWrite — Used to write the converted Markdown to your clipboard.

Third-party services

None. The extension uses a bundled copy of the Turndown library (MIT licensed) and makes no external network requests.

Changes to this policy

If anything ever changes, we'll update this page and the date above.
